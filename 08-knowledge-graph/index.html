Cose 力导向 Compound Spring Embedder

把节点看成互相排斥的小球,把边看成弹簧。系统运行一会儿后会达到相对稳定的位置。

适合关系网探索,能自然把相关实体拉近。
Breadthfirst 层次布局 Breadth-first Layout

按层级一层层排开,像组织架构图或树状目录。

适合有明显父子层级或上下游关系的数据。
Circle 圆周布局 Circle Layout

所有节点排在圆上,关系线在内部连接。

适合节点不多、想快速看整体连接时。
Concentric 同心布局 Concentric Layout

重要或连接多的节点靠中心,其他节点按圈层向外排。

适合突出核心实体和边缘实体。
