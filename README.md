# 📊 学术可视化图表全景手册 · Vis Handbook

> 系统整理 **32 类**专业级可视化图表,每张图配有 **可交互 Demo** + **原理** + **数据场景** + **二创思路**。
> 面向想深入可视化方向、并在现成框架上二次开发的同学。

---

## 🚀 快速开始

**直接打开**:`index.html`(用 Chrome / Edge / Firefox 现代浏览器)

- 封面页是 32 张图的卡片目录,点击任意卡片进入详情
- 详情页左侧是**可交互 Demo**(嵌入 iframe),右侧是**原理/场景/二创**说明
- 底部翻页栏 + 键盘 ← → 切换图表
- 也可单独打开每个子目录的 `index.html` 全屏体验

**零依赖**:所有图表库通过 CDN 加载,无需 npm install。双击 HTML 即可运行。

## 💡 二创指南

每张图的详情页都列了 4 条具体的二创思路。优先推荐:

1. **DTW + 聚类** → 时序相似度分析(学术性最强)
2. **降维图 + 联动** → ML 数据探索工具
3. **知识图谱 + LLM** → 自然语言图查询(当下最热)
4. **山脊图 + 动画** → 数据新闻叙事(美学强)
5. **切尔诺夫脸 + 3D** → 数据艺术装置
6. **Voronoi + 加权** → 设施选址决策系统

完整建议见 [`resources.md`](./resources.md)。
